# pap-engine
game engine untuk kita belajar bareng

## Cara Menjalankan

```bash
# clone repository
https://github.com/bellshade/pap-engine

# build project
cd pap-engine
# buat folder build
mkdir build
cd build
# compile dan jalankan
cmake ..
make

# jalankan programnya
./papEngine
````
